# quick-commerce-powerbi-dashboard
Interactive Power BI dashboard analyzing quick commerce sales performance 
Quick Commerce Sales Dashboard (Power BI)

 Overview

This project presents a fully interactive Power BI dashboard designed to analyse quick-commerce retail sales performance across products, outlets, and locations. The dashboard was built to simulate a real-world business reporting solution where stakeholders can monitor KPIs, explore outlet performance, and identify revenue-driving product categories.

---

 Business Problem

Quick-commerce businesses handle large product inventories across multiple outlet locations.
To support operational decisions, they require a consolidated reporting dashboard that can:

- Track overall sales and product distribution
- Identify high-performing outlet locations and sizes
- Analyse product category contribution to revenue
- Monitor outlet establishment trends over time
- Enable fast drill-down from summary metrics to detailed outlet-level insights

This dashboard addresses these needs through interactive reporting and structured data modelling.

---

 Key Dashboard Capabilities

* Advanced Interactivity

- Implemented page navigation buttons for smooth multi-page dashboard flow
- Enabled drill-through functionality to allow users to move from summary visuals to detailed outlet-level analysis
- Designed custom tooltips to surface additional metrics without overcrowding visuals

* KPI Monitoring

Key performance indicators displayed include:

- Total Sales
- Number of Items
- Average Sales
- Average Customer Rating

These provide a quick operational snapshot for business monitoring.

* Dynamic Filtering & Segmentation

Interactive slicers allow real-time filtering by:

- Outlet Location Tier
- Outlet Size
- Item Type

Cross-filtering ensures that all visuals update dynamically to support exploratory analysis.

---

 Analytical Insights Supported

The dashboard enables identification of:

- Revenue contribution differences across outlet tiers
- Sales performance variation by outlet size
- Category-level sales dominance patterns
- Historical sales distribution by outlet establishment year
- Inventory and rating distribution trends

---

 Technical Implementation

- Data cleaning and transformation performed using Power Query
- Data modelling structured to support efficient aggregation and filtering
- Visual layout designed for high readability and business usability
- Performance optimised to ensure smooth interaction with multiple filters and drill actions
